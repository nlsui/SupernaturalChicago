# Nur-Für-Spielleiter

Dieses Verzeichnis enthält Wissen, das ausschließlich dem Spielleiter zugänglich ist.

---

## Fälle

Der Spielleiter entwirft neue Fälle für die Spieler. Jeder Fall wird hier vollständig dokumentiert – inklusive Plot, beteiligter Orte, Nicht-Spieler-Charaktere, Encounter und allem weiteren relevanten Material.

**Vorlage:** `Fälle/_TEMPLATE.md`

---

## Fraktionen, Kreaturen, Nicht-Spieler-Charaktere, Spezielle-Orte, Übernaturliches-Wissen

Diese Ordner enthalten Weltwissen, das nur dem Spielleiter bekannt ist. Finden die Spieler im Laufe des Spiels Informationen darüber heraus, kann der Spielleiter eine abgespeckte Kopie des entsprechenden Eintrags im jeweiligen Gegenstück in `Spieler-Wissen` anlegen – mit ausschließlich den Informationen, die die Spieler tatsächlich in Erfahrung gebracht haben.

Jeder Ordner enthält eine Vorlage für solche abgespeckten Public-Knowledge-Einträge:
